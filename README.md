# AutoSelect
自动选择excel进行数据读取、以数量抽选数据、并导为excel文件
在最后一个选择按钮“选择目标数据”时，选择的是列中去重后的数据，选择完毕，会在弹出的界面中显示所有包含选择字段的行数据
弹出的显示框中，第一行是excel数据的第一行数据，一般是列名，第二行及以后行是数据
